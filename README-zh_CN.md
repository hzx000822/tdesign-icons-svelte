TDesign icons svelte是一个图标库。
使用了tdesign-icons中的svg文件

# 📦 安装

```shell
npm i tdesign-icons-svelte
```

# 🔨 使用

```js
// svgsprite
import { Icon } from 'tdesign-icons-svelte';

<Icon name="add" />;

// single icon
import { CloseIcon, ChatIcon } from 'tdesign-icons-svelte';

<CloseIcon />;
```
